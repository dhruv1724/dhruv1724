## Featured Project

### SiteBuilder – AI Powered Website Generator (PERN Stack)

A full-stack SaaS application that generates complete, production-ready websites using AI.

**Core Features:**
- AI-generated Tailwind CSS websites
- Prompt enhancement and AI response processing
- Project-based version control system
- Live preview rendering
- Publish and sharing functionality
- Stripe Checkout integration
- Stripe Webhook handling
- Credit-based AI usage system
- Authentication and session management

**Architecture Flow:**
1. User authentication  
2. Prompt submission  
3. AI enhancement and processing  
4. HTML + Tailwind CSS generation  
5. Version storage in PostgreSQL  
6. Preview / publish / rollback functionality  
7. Credit deduction per generation
